# Parkinson Disease Prediction

A data science project focused on building a complete machine learning pipeline using a Parkinson's disease dataset.

The main goal is to explore the dataset, develop predictive machine learning models, expose predictions through an API, and build an interactive dashboard for data visualization.

## Dataset

The dataset contains clinical information and acoustic voice measurements collected from patients diagnosed with Parkinson's disease.

Some of the main features include:

- Age
- Sex
- Motor UPDRS
- Total UPDRS
- Jitter
- Shimmer
- HNR
- NHR
- RPDE
- DFA
- PPE
